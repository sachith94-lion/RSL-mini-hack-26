# UI/UX Specification

## Theme
- **Primary Color:** `#10b981` (Success Green) for habit completion.
- **Background:** Dark Mode (Deep Navy/Black) for a premium "health app" feel.

## Layout
- **Mobile:** Bottom Navigation (Dashboard, Habits, Settings).
- **Checklist:** Large tap targets (44px+) for "checking off" habits.
- **Feedback:** Haptic-style visual feedback (scale animations) when a habit is completed.
