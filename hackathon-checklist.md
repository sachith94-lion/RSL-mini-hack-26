# 90-Minute Hackathon Checklist

## T-Minus 90: Setup (15m)
- [ ] Initialize Angular 18 project.
- [ ] Install Tailwind CSS & DaisyUI.
- [ ] Configure Firebase Console (Project, Auth, Firestore).
- [ ] Run `ng add @angular/fire`.

## T-Minus 75: Core Logic (30m)
- [ ] Implement `AuthService` (Login/Logout).
- [ ] Create `HabitService` with Signals for Habit CRUD.
- [ ] Build the "Add Habit" form/modal.

## T-Minus 45: Visualization (30m)
- [ ] Build the "Today" view (Checklist).
- [ ] Integrate `chart.js` or `ng2-charts` for the Dashboard.
- [ ] Implement "Completion Rate" logic.

## T-Minus 15: Deployment & Docs (15m)
- [ ] Run `ng deploy` to Firebase Hosting.
- [ ] Finalize `README.md` and AI Workflow documentation.
- [ ] Record a 30-second loom/gif of the working app.
