# Product Requirements Document (PRD) - HabitPulse

**Product:** HabitPulse — Daily Habit & Health Tracker
**Status:** Hackathon Draft (90-min Sprint)

## 1. Overview
A lightweight, mobile-first habit tracker that allows users to log daily activities (water, exercise, sleep) and visualize their consistency.

## 2. Functional Requirements
- **FR-01: Auth:** Simple Email/Password login via Firebase.
- **FR-02: Habit Management:** Users can create habits with categories (Health, Productivity, etc.).
- **FR-03: Daily Logging:** A "Checklist" view for today's habits.
- **FR-04: Progress Dashboard:** Visual charts showing 7-day consistency and category breakdown.
- **FR-05: Persistence:** All data stored in Firestore, isolated by User ID.

## 3. Success Metrics
- Time to log a habit: < 5 seconds.
- Mobile Score: 100% responsive on small screens.
