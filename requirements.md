# Requirements - HabitPulse

## 1. Feature Scope

### Core (P0 - MVP)
- **User Identity:** Simple login/signup via Firebase Auth.
- **Habit Logging:** Track water, exercise, study, sleep, etc.
- **Categories:** Group habits by Health, Fitness, Productivity, or Upskilling.
- **Daily Checklist:** A mobile-responsive "Today" view to check off habits.
- **Basic CRUD:** Create, edit, and delete habit definitions.

### Analytics (P1)
- **Today’s Summary:** Completion percentage and key stats on landing page.
- **Dashboard:** 7-day consistency trends using charts.
- **Category Breakdown:** Visual distribution of habits.

## 2. Technical Constraints
- **Framework:** Angular 18 (Standalone Components + Signals).
- **Backend:** Firebase Firestore (Real-time DB).
- **Hosting:** Firebase Hosting (Free Tier).
- **UI:** Tailwind CSS + DaisyUI for rapid, responsive styling.
